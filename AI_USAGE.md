## 功能模块

### 1. 登录/注册页面
- 用户可以注册新账号（手机号、密码、姓名）
- 已有账号可以直接登录
- 数据存储在本地 localStorage 中

### 2. 身份选择页面
- 用户可以选择"我想点外卖"或"成为外卖员"
- 两种角色对应不同的功能界面

### 3. 点餐界面（用户端）
- 展示各类美食列表（快餐、饮品、甜点、小吃）
- 支持分类筛选和搜索功能
- 购物车功能（添加/删除商品）
- 订单结算功能

### 4. 接单界面（骑手端）
- 展示待接订单列表
- 显示订单详情（商品、地址、联系方式）
- 一键接单功能

### 5. 个人界面
- 显示用户信息（头像、昵称、手机号）
- 订单统计（待付款、待接单、配送中、已完成）
- 钱包余额、评价、帮助中心、设置
- 退出登录功能

## 技术特点

- **响应式设计**：适配竖屏移动端显示（最大宽度 420px）
- **UI框架**：使用 TailwindCSS 实现现代化界面
- **图标**：使用 Material Icons
- **数据持久化**：使用 localStorage 保存用户信息和购物车数据
- **模拟数据**：内置美食和订单数据，无需后端即可完整体验

## 步骤一：创建 GitHub 仓库

   - Description: 可选，描述项目
   - 选择 **Public**（公开仓库）
   - 勾选 **Add a README file**
   - 点击 **Create repository**

## 步骤二：上传项目文件

### 方法一：使用 GitHub 网页上传

1. 进入你的仓库页面
2. 点击 **Add file** -> **Upload files**
3. 拖拽 `index.html` 和 `app.js` 到上传区域
4. 点击 **Commit changes**

### 方法二：使用 Git 命令（推荐）

1. 在 VS Code 中打开终端（按 `Ctrl+`）
2. 初始化 Git：
   ```bash
   git init
   ```
3. 配置用户名和邮箱：
   ```bash
   git config user.name "你的GitHub用户名"
   git config user.email "你的GitHub邮箱"
   ```
4. 添加文件：
   ```bash
   git add .
   ```
5. 提交：
   ```bash
   git commit -m "Initial commit"
   ```
6. 关联远程仓库（替换为你的仓库地址）：
   ```bash
   git remote add origin https://github.com/你的用户名/你的仓库名.git
   ```
7. 推送代码：
   ```bash
   git push -u origin main
   ```

## 步骤三：配置 GitHub Pages

1. 进入仓库页面，点击 **Settings**
2. 滚动到 **Pages** 选项
3. 在 **Source** 部分：
   - Branch: 选择 `main`
   - Folder: 选择 `/root`
   - 点击 **Save**
4. 等待几分钟后，页面会显示你的网站链接（例如 `https://你的用户名.github.io/你的仓库名/`）
